CSV Format Instructions

Required Columns:

  • name - Contact's full name (required)
  • email - Email address (required)

Optional Columns:

  • phoneNumber - Phone number
  • address - Physical address
  • description - Notes about the contact
  • websiteLink - Website URL
  • linkedInLink - LinkedIn profile URL
  • favorite - true/false or 1/0

Example CSV Format:

name,email,phoneNumber,address,favorite
John Doe,john@example.com,+1234567890,123 Main St,true
Jane Smith,jane@example.com,+0987654321,456 Oak Ave,false
